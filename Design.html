Justification des couleurs

La couleur principale des éléments de la page web est le noir, représentant les soupers. Tous les éléments noirs sont transparents à 50%, pour éviter de rendre les pages fades et ennuyeuses pour l'oeil. La couleur du texte est le blanc. Nous avons choisi ces deux couleurs car elles sont absolument opposées et ce contraste fait ressortir davantage l'élément importante du site : les informations.

Toutefois, la principale source de couleur provient des images d'arrières plans. Sur la plupart des pages, les images sont foncées pour mettre l'accent sur l'information contenue dans la page plutôt que la page en elle-même. Cependant, la page "Promotions" a, au contraire, un fond très pale. Cette image a été choisie pour rendre la page plus légère, puisqu'une page de promotions devrait contenir de bonnes nouvelles et le blanc représente parfaitement cette pensée. Aussi, du rouge est utilisé dans la barre de navigation. Cette couleur a été choisie pour rappeller le feu d'un barbecue. Cette couleur n'est toutefois présente que pour de minces lignes afin de ne pas trop aggresser l'utilisateur.

Étapes réalisées

  1. Objectifs du site : Le site est cohérent avec lui-même. En effet, toutes les pages suivent la même forme. Il sert également à faire connaître Coq Coq Cadet. Nous avons d'ailleurs ajouté des adresses et numéros de téléphones (évidemment fictifs) pour améliorer la crédibilité.
  2. Type de site : Incitatif. Le site sert à divulguer de l'information pour que l'internaute veuille venir et consommer au restaurant Coq Coq Cadet.
  3. Architecture des informations : Séparations en pages distinctes qui renferment chacune une partie distincte des informations transmises par le site (sans répétition). L'accès à l'information est facile, grâce à la barre de navigation pertinente. Les actions à accomplir sont clairement définies (mise en évidence des items de la naviagtion lorsque l'utilisateur promène sa souris dessus, choix pour certaines options du formulaire...)
  4. Scénarisation : Présentation plate sommaire. Toutes les informations pertinentes sont divulguées de façon brève, mais efficace. L'ensemble du site divulgue les informations de façon plate. Toutefois, la page de réservation demande des actions de l'utilisateur. Également, toutes les pages du site suivent la même forme.
  5. Contraintes d'usage : Navigation arborescente. L'ensemble des informations du site sont liées à la branche principale (Coq Coq Cadet) et se divise en plusieurs branches ("Accueil", "Menu", "Réservation", "Promotion"). Le site a une version pour mobile et tablette.

Accessibilité

Dans l'ensemble du site, aucune image n'est d'importance primordiale pour la transmission d'information. Aussi, toutes les images ont une description si elles n'apparaissent pas en raison d'une connexion lente ou d'une incompatibilité. Pour les images de fond, les attribute "alt" sont vide, pour que la présentation en cas d'absence de l'image reste esthétique. Toutes les images du menu sont de plus accompagnées d'une description, pour pouvoir transmettre l'information même sans les images.

Le site est également prévu pour accommoder les utilisateurs d'appareils mobiles. Lorsque la page est générée dans une fenêtre de largeur inférieure ou égale à la 1024px (dimension typique d'une tablette), la barre de navigation change pour devenir un menu déroulant et le logo se place au centre de la page. Cela permet une navigation plus fluide. Lorsque la page est générée dans une fenêtre de largeur inférieure ou égale à 480px (dimension typique d'un téléphone intelligent), la barre de navigation rapetisse, et les proportions des lettres grandissent, pour permettre une meilleure lecture des informations.

Également, presque toutes les images sont réduites avant d'être envoyées vers le site Web pour accomoder les visiteurs avec une connexion internet plus lente. Seules les images de fond n'ont pas été préréduites, car elles doivent rester le plus claires possible pour n'importe quelle taille d'écran.

Le formulaire est aussi accessible, car les options peuvent être sélectionnées à l'aide du clavier ou de la souris.
